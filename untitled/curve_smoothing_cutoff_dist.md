## _Lissage des courbes_ Distance maxi

### Leírás

Distance maximale entre deux points pour permettre l'ajout de nouveaux points de lissage. Permet d'éviter de distordre les longues zones droites.

> 0 pour désactiver.

* Valeur par défaut : 2mm
