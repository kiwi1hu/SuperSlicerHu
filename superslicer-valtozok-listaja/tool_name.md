## Nom extrudeuse

### Leírás

Utilisé uniquement pour klipper, où vous pouvez nommer l'extrudeuse. S'il n'est pas défini, ce sera **'extruderX'** avec **'X'** remplacé par le numéro de l'extrudeuse.
